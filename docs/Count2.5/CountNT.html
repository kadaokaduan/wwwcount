Count Release 2.3 for NT (intel)

Latest Release: May 05, 1996
Initial Release: March 10, 1996

What's new

This is the 2.3 final binary release of the WWW homepage access counter for NT (intel). Please look at the page for Count 2.3 for source distribution and detail documentation.

This version has all of the features of the unix version.

Download

The binary distribution for NT is available as a zipped file (wwwcount.zip).

Download the archive (wwwcount.zip)

After downloading, unzip the archive in your cgi-bin, scripts (or wherever you keep CGI programs) using the "-d" switch. For example: 

    unzip -d wwwcount.zip
If you use winzip or some other program which maintains the directory structure in the zip file, you don't have to worry about -d switch.

The unzipping procedure will create the following files: 

    README (file)
    Count.exe (executable file)
    mkstrip.exe (executable fiel)
    wcount (directory)
Use File Manager to copy or move the directory wcount to C drive.

cd to c:\wcount\conf and edit the file count.cfg. The file is heavily commented. Read the counter main page for details.

Read the counter main page and look at the Example page for help. In the example page, the name of the counter program is Count.cgi, for NT the name will be Count.exe.
